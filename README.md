# 🛒 Nossa Lista

Aplicação web simples para gerenciar uma lista de compras compartilhada em tempo real.

## ✨ Funcionalidades

- Adicionar itens à lista
- Definir quantidade (opcional)
- Marcar itens como comprados
- Remover itens
- Estatísticas da lista (total, pendentes e comprados)
- Interface moderna e responsiva
- Identificação de usuário simples

## 🛠 Tecnologias

- PHP
- JavaScript
- HTML5
- CSS3

## 💡 Objetivo

Projeto criado para facilitar o compartilhamento de listas de compras entre pessoas, com uma interface simples, rápida e fácil de usar.
